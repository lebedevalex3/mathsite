# Skills-First UI Wireframe

Цель: перейти от “учебника” к продукту, где основная единица прогресса — навык.

## Экран 1: Главная (skills-first)

```text
┌────────────────────────────────────────────────────────────────────┐
│ HEADER: Лого | Темы | Рейтинг | Бейджи | Учителю                  │
├────────────────────────────────────────────────────────────────────┤
│ HERO                                                              │
│ "Прокачивай навыки по математике шаг за шагом"                    │
│ [ Я ученик ]  [ Я учитель ]                                       │
│ Подпись: "Учителю — собрать работу, ученику — тренировать навык"  │
├────────────────────────────────────────────────────────────────────┤
│ CONTINUE (персонально)                                            │
│ Тема: Пропорции | Освоение: 42% | Следующий: "Найти средний..."   │
│ [ Продолжить навык ]   [ Повторить слабые места ]                 │
├────────────────────────────────────────────────────────────────────┤
│ ТЕМЫ (карточки)                                                    │
│ ┌ Тема: Пропорции      ┐ ┌ Тема: Уравнения      ┐                 │
│ │ 10 навыков           │ │ 8 навыков            │                 │
│ │ 4 освоено            │ │ 2 освоено            │                 │
│ │ Следующий: ...       │ │ Следующий: ...       │                 │
│ │ [Открыть карту]      │ │ [Открыть карту]      │                 │
│ └──────────────────────┘ └──────────────────────┘                 │
├────────────────────────────────────────────────────────────────────┤
│ МОТИВАЦИЯ                                                          │
│ Уровень 5 | 320 XP | Серия 6 дней | До бейджа "Точность": 3 шага  │
│ [Смотреть все бейджи]                                              │
└────────────────────────────────────────────────────────────────────┘
```
