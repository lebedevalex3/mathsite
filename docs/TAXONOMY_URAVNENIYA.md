- `topic_id`: `g5.uravneniya`

## Skill IDs (операции)

- `g5.uravneniya.ponyat_uravnenie_i_koren` — понимать, что такое уравнение и корень уравнения
- `g5.uravneniya.proverit_koren` — проверять, является ли число корнем уравнения
- `g5.uravneniya.reshat_x_plus_minus_a_ravno_b` — решать уравнения вида x + a = b и x - a = b
- `g5.uravneniya.reshat_a_plus_minus_x_ravno_b` — решать уравнения вида a + x = b и a - x = b
- `g5.uravneniya.reshat_mnozhenie_i_delenie` — решать уравнения вида a·x = b и x : a = b
- `g5.uravneniya.reshat_prostye_tekstovye_uravneniya` — составлять и решать простые уравнения по условию
