# Obsidian Vault

Этот каталог предназначен для Obsidian как рабочее хранилище контентного pipeline.

Назначение vault:
- планирование тем и навыков;
- карта связей между навыками;
- идеи листов и маршрутов;
- рабочие заметки по дефицитам и следующим шагам.

Vault не является источником публикации сайта.

Источником публикации остаются:
- `data/tasks/` для банка задач;
- `content/seo/` для готовых SEO-материалов;
- `latex-math-sheets/` для производственных LaTeX-шаблонов и PDF.

Базовые правила:
- одна тема = одна папка;
- файл навыка называется ровно по `skill_id`;
- задачи не хранятся в Obsidian;
- в Obsidian хранятся смысл, связи, маршруты и editorial planning.

## Как использовать ссылки из vault

Внутри заметок есть обычные Markdown-ссылки на реальные файлы репозитория:
- `data/tasks/...` для банка задач;
- `docs/TAXONOMY_...` для таксономии;
- `content/seo/...` для SEO-витрины;
- `src/...` для runtime-кода и topic metadata.

Если ссылка ведет на большой JSON-файл, ориентируйтесь по `skill_id` из заметки и ищите его в файле.
